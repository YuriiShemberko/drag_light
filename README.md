# Drag racing start lights control project

How it works: https://www.youtube.com/shorts/FX69eCEJkKs

## Hardware prereqs:
- Arduino Board (Nano, e.g.)
- 3 x 12V relays
- 3 x 12V lights (e.g 
- 433Mhz RC module
- Any push button (optional)

## Software
- Arduino IDE

## Assembling
### Element - Arduino PIN number (mode)

Top Light Relay - PIN #3 (out) <br>
Middle Light Relay - PIN #4 (out) <br>
Bottom Light Relay - PIN #5 (out) <br>

Remote Control - PIN #7 (in) <br>
Push Button - PIN #2 (in) <br>
